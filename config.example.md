## Evidence tags

Tag each quote with the type that best fits:

- `qualitative` — opinions, pain points, wishes, descriptions of workflows
- `quantitative` — numbers, metrics, frequencies, durations, costs
- `feature-request` — explicit asks for new functionality
- `pain-point` — frustrations with current tools or processes
- `workflow` — descriptions of how they do things today
- `buying-signal` — willingness to pay, budget mentions, urgency
